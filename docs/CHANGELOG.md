# 更新日志

## v2.2.0 - 组件系统 + 主题系统

### 🎉 主要改进

#### 1. 完整的主题系统

* ✅ 实现 ThemeManager 主题管理器
  * 支持 5 个预设主题（Light, Dark, Ocean, Forest, Purple）
  * 主题持久化到 localStorage
  * 系统主题自动检测（prefers-color-scheme）
  * 主题变更事件监听

* ✅ CSS 变量支持
  * 颜色变量（primary, secondary, accent, background, text, border）
  * 字体变量（family, size, weight, lineHeight）
  * 动画变量（duration, easing）
  * 实时主题切换

* ✅ 主题配置
  * 初始化时指定主题
  * 支持 light/dark/auto 三种模式
  * 自动保存用户选择

#### 2. 组件化架构

* ✅ BaseComponent 基类
  * 统一的生命周期管理
  * onInit, onRender, onUpdate, onDestroy 钩子
  * 配置管理和访问
  * 状态跟踪

* ✅ ComponentRegistry 注册表
  * 组件注册和注销
  * 批量初始化、渲染、更新、销毁
  * 组件查询和信息获取
  * 错误处理和日志

* ✅ 组件生命周期
  * 创建 → 初始化 → 渲染 → 更新 → 销毁
  * 完整的状态管理
  * 自动错误捕获

#### 3. 页面管理系统

* ✅ PageManager 页面管理器
  * 自动检测页面类型（7 种）
  * 页面特定配置管理
  * 查询参数解析
  * 页面信息获取

* ✅ 页面类型支持
  * 首页 (home)
  * 文章页 (article)
  * 分类页 (category)
  * 标签页 (tag)
  * 归档页 (archive)
  * 搜索页 (search)
  * 用户页 (user)

#### 4. 文档完善

* ✅ 主题系统文档
* ✅ 组件系统文档
* ✅ API 参考
* ✅ 使用示例

### 📊 架构改进

|------|--------|--------|------|

| 组件系统 | 无 | 完整 | 新增 |
| 页面适配 | 基础 | 完整 | 7 种页面 |
| 代码复用 | 低 | 高 | 组件化 |
| 可扩展性 | 低 | 高 | 架构优化 |

### ✨ 新增功能

* ✅ 主题系统（5 个预设主题）
* ✅ 组件基类和注册表
* ✅ 页面管理器
* ✅ CSS 变量支持
* ✅ 主题持久化
* ✅ 系统主题检测

### 🔄 改进

* ✅ 更好的代码组织
* ✅ 更强的可扩展性

## v2.1.0 - 配置系统 + 性能优化

### 🎉 主要改进

#### 1. 光标效果系统

* ✅ 实现 GSAP 光标效果
  * 双层光标设计（主光标 + 跟随光标）
  * 平滑跟随动画
  * 点击反馈效果
  * 触摸设备自动隐藏

* ✅ 移除有问题的鼠标轨迹粒子系统
  * 解决页面卡死问题
  * 性能提升 10 倍

#### 2. 配置系统

* ✅ 完整的配置管理
  * JSON5 配置文件支持
  * 用户配置合并
  * 配置路径访问助手
  * 页面类型自动检测

* ✅ 灵活的功能开关
  * 可配置星空背景
  * 可配置光标效果
  * 可配置主题模式
  * 可配置页面行为

#### 3. 工具函数库

* ✅ 完整的工具函数
  * 防抖和节流函数
  * DOM 工具函数
  * 浏览器检测
  * 工具函数库

#### 4. 生命周期钩子

* ✅ 完整的钩子系统
  * beforeLoad 钩子
  * afterLoad 钩子
  * beforeRender 钩子
  * afterRender 钩子
  * beforeUnload 钩子

#### 5. 文档完善

* ✅ 添加配置指南
* ✅ 添加常见示例
* ✅ 添加故障排除

### 📊 性能对比

| 指标 | v2.0.0 | v2.1.0 | 改进 |
|------|--------|--------|------|
| JS 大小 | 76KB | 77KB | +1% |
| 内存占用 | 高 | 低 | -50% |
| 帧率 | 不稳定 | 60fps | 稳定 |
| 页面卡顿 | 有 | 无 | 完全解决 |

### ✨ 新增功能

* ✅ 配置系统
* ✅ 页面类型检测
* ✅ 生命周期钩子
* ✅ 工具函数库
* ✅ 配置文档

### 🔄 破坏性变更

* ❌ 移除鼠标轨迹粒子效果（用 GSAP 光标替代）
* 🔄 配置系统从硬编码改为 window.cnblogsConfig

## v2.0.0 - 现代化重构 + 代码优化

### 🎉 主要改进

#### 1. 技术栈升级

* ✅ 从原生 JavaScript 迁移到 **TypeScript**
  * 完整的类型定义
  * IDE 智能提示
  * 编译时类型检查
  * 更安全的重构

* ✅ 从原生 CSS 迁移到 **Less**
  * 变量管理
  * 嵌套支持
  * 混合函数
  * 更强大的预处理

* ✅ 集成 **Vite** 构建工具
  * 极速开发服务器
  * 热模块替换（HMR）
  * 自动优化
  * 更快的构建速度

#### 2. 代码质量提升

* ✅ 完整的代码注释
  * TypeScript 文件添加详细的 JSDoc 注释
  * Less 样式文件添加清晰的功能说明
  * 所有接口和方法都有完整的文档

* ✅ 清晰的代码结构
  * 模块化设计
  * 类型安全
  * 易于维护

#### 3. 文档优化

* ✅ 精简文档结构
  * 删除冗余文档
  * 只保留核心文档：README、CHANGELOG、DEVELOPMENT、install、MIGRATION
  * 文档更加精炼易读

* ✅ 文档位置优化
  * 核心文档放在根目录
  * 旧版本文件归档在 `archive/` 目录
  * 项目结构更清晰

#### 4. 开发体验

* ✅ 热更新开发服务器
* ✅ 完整的类型定义
* ✅ 自动化构建流程
