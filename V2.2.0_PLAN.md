# StarryNebula v2.2.0 开发计划

## 版本目标

**主题：** 功能扩展和构建优化  
**目标：** 实现完整的主题系统、组件化架构和构建优化

---

## 核心任务

### 1. 主题系统实现

#### 1.1 主题配置扩展
- 创建 `src/themes/` 目录
- 实现多个预设主题（light, dark, ocean, forest 等）
- 每个主题包含：颜色方案、字体、动画配置

#### 1.2 主题切换功能
- 实现主题持久化（localStorage）
- 支持系统主题检测（prefers-color-scheme）
- 实现主题切换动画

#### 1.3 主题 API
```typescript
// 使用示例
const themeManager = new ThemeManager()
themeManager.setTheme('dark')
themeManager.toggleTheme()
themeManager.onThemeChange((theme) => {
  console.log('Theme changed to:', theme)
})
```

### 2. 组件化重构

#### 2.1 组件结构
```
src/components/
├── Starfield/
│   ├── Starfield.ts

│   └── types.ts
├── Cursor/

├── Sidebar/
│   ├── Sidebar.ts
│   ├── sidebar.less
│   └── types.ts
└── index.ts
```

#### 2.2 组件基类
```typescript
// src/components/BaseComponent.ts
export abstract class BaseComponent {
  protected config: any
  protected element: HTMLElement | null = null

  abstract init(): void
  abstract destroy(): void
  abstract render(): void
}
```

#### 2.3 组件注册系统
```typescript
// src/components/ComponentRegistry.ts
export class ComponentRegistry {
  private components: Map<string, BaseComponent> = new Map()

  register(name: string, component: BaseComponent): void
  unregister(name: string): void
  get(name: string): BaseComponent | undefined
  getAll(): BaseComponent[]
}
```
